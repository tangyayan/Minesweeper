# Minesweeper

---

大一做的一个简易扫雷游戏

### 项目简介

用cpp写的一个扫雷游戏

### 快速开始

需要安装easyx的图形库和vscode2022

然后运行 `testeasyx\testeasyx.sln`文件即可

### 使用说明

1、用户首先在开始界面选择“开学模式·易”、“期中周模式·中”或者“期末周模式·难”，      开始游戏。

2、在游戏界面中使用鼠标左键点开格子，使用鼠标右键标记“ddl”，再次点击右键可取消标记。

3、每个格子显示的数字即以该格子为中心的九宫格中所包含的“ddl”个数,用户通过推理判断出“ddl”的位置。

4、若用户使用左键点开“ddl”，则游戏失败；若用户将所有“ddl”正确标记，则游戏胜利。

5、在游戏胜利和游戏失败界面，用户可选择“再来一局”或者“退出游戏”。
